# Student Task Manager

## Problem
Students often forget their assignments and tasks. Managing tasks manually can be messy.

## Solution
This project is a simple Java command-line application that allows students to manage tasks.

## Features
- Add tasks
- View tasks
- Mark tasks as completed

## Technologies Used
Java
ArrayList
Object-Oriented Programming

## How to Run

Compile:
javac Task.java TaskManager.java

Run:
java TaskManager
